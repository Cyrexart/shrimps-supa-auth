## Getting Started

First, download or clone the repository with this command:

```bash
git clone https://github.com/Cyrexart/shrimps-supa-auth my-app
cd my-app
``` 

Install all dependencies:

```bash
npm install 
# or pnpm install / yarn install
```

Create a new project at supabase.com, rename the `.env.example` file to `.env.local`, and fill it in with your details.

Run Development Server

```bash
npm run dev
# or pnpm dev / yarn dev 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
